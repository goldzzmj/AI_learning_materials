# AI Learning Materials

本项目用于记录和分享 AI 相关的学习资料与实践项目。

## 目录结构

### 课程一：Mini-vllm
位于目录 `课程一_mini-vllm/` 下。
- **核心内容**：基于 mini-vllm 的完整学习指南。
- **文件**：`mini-vllm完整学习指南.md`
- **学习目标**：深入理解 vLLM 推理框架的核心原理与实现。

### 课程二：Sglang-Diffusion
位于目录 `课程二_sglang-diffusion/` 下。
- **核心内容**：Sglang 框架在 Diffusion 模型中的应用与分析。
- **目录**：`sglang_diffusion_analysis`
- **学习目标**：探索结构化生成语言（Structured Generation Language）如何优化扩散模型的控制与生成。


### 课程三：VLM_Training
位于目录 `课程三_VLM_Training/` 下。
- **核心内容**：基于 ms-swift 框架的多模态大模型（VLM）训练与微调完全指南，以 Qwen3-VL 为例进行实战解析。
- **文件**：`VLM_Training_ms_swift_Guide.md`
- **学习目标**：掌握 VLM 的全流程训练方法（SFT, DPO, GRPO 等）及 ms-swift 框架的核心原理与应用。


### 课程四：Agent Skills
位于目录 `课程四_agent_skills/` 下。
- **核心内容**：Agent Skills 方法全面技术解析，涵盖架构设计、路由算法、编排机制、生态系统及最佳实践。
- **文件**：`agent_skills_comprehensive_guide.md`
- **学习目标**：深入理解智能体技能（Agent Skills）的模块化设计范式，掌握 Skill Router、Orchestrator 等核心组件的原理与实现。
